Cart Testing

Open the browser console and check the network tab to debug cart issues:

  1. Go to the products page
  2. Try adding a product to cart
  3. Open the cart sidebar
  4. Check if images load properly
  5. Test quantity increase/decrease buttons
  6. Check console for any errors

Debug Steps:

  • Check if product images exist in /public/ folder
  • Verify cart context is properly initialized
  • Test localStorage persistence
  • Verify button event handlers
